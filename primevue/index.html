PrimeVue Theme Designer

PrimeVue is a design-agnostic library and the theming system is based on the Theme Designer, the official tool to create themes for the components.

Designer includes a SASS enabled infrastructure that has over 500 customizable variables, a live editor and a Vue-CLI application to test your themes with ease along with built-in theme samples that are embedded in PrimeVue Core.

Check out the Theme Designer homepage to learn more about the features and interact with the live editor.
